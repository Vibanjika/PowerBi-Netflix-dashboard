# PowerBi-Netflix-dashboard
An interactive Power BI dashboard analyzing Netflix movies and TV shows. It visualizes total shows, country distribution, genre trends, release year growth, ratings, and Movies vs TV Shows comparison. Built using Power BI for insightful data visualization.
